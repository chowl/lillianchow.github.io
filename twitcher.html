      Final project for Girls Who Code. Takes the Twitter API and pulls all the tweets from an entered Twitter handle. Then uses the selected filter which is programmed with Regular Expressions to filter through the data. All tweets containing words from the filter will then be outputted with the word it was flagged for at the end. The top will also state how many times the inputed user was flagged for the filter. (Python, HTML, CSS, JavaScript, RegEx, Bootstrap, Flask)
